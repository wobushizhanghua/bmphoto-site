RT Slider is a Simple Plugin which Helps you in Creating a Slider for your Front Page or Homepage. Works with Supported Themes only.

== Description ==

<strong>Developers:</strong> Integrate this Plugin with your themes.
<a href="https://github.com/rohitink/rt-slider" target="_blank">Documentation here</a>

RT Slider is a Very Simple and Easy to use Plugin. All this Plugin does is that, it adds a Panel to your WordPress Customizer, which helps you create a Slider. It works with only those themes, which support this plugin and does not work with every theme.

The Benefit of this Plugin, is that you don't loose your Slider data once you switch from one theme to another.

== Installation ==

This Plugin is Generally Recommended by themes which Support it. If you theme doesn't support this plugin, you will get a notice as soon as your activate this plugin.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Once its Activated, Go to Appearance > Customize to Configure the Slider.
